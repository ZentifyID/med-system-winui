# Med System (WinUI 3)

Приложение для медпункта колледжа: учёт обращений, данных сотрудников
и студентов, лекарств. Переписано с Python/tkinter на C#/WinUI 3.

## Структура решения

```
MedSystem/
├── MedSystem.Core/    Домен: модели, правила сроков, валидация (без зависимостей)
├── MedSystem.Data/    Слой данных: SQLite (Microsoft.Data.Sqlite), репозитории
└── MedSystem.App/     Интерфейс: WinUI 3, страницы и формы
```

Зависимости слоёв: `App → Data → Core`.

## База данных

SQLite, файл `med_system.db` в LocalFolder приложения (путь показан
на главной странице). Схема полностью совместима с python-версией
([med-system](https://github.com/ZentifyID/med-system)) — обе программы
могут открывать один и тот же файл.

## Сборка

Visual Studio 2022 с рабочей нагрузкой «Разработка приложений для Windows».
Открыть `MedSystem/MedSystem.slnx`, F5. NuGet-пакеты подтянутся автоматически.

## Возможности

Сотрудники и студенты (с медицинскими сроками: санминимум, медосмотр,
флюорография), учебные группы с переводом на следующий курс, лекарства
с контролем сроков годности и остатков, заказ партий, журнал обращений
с автодополнением отправителя и диагнозов по справочнику МКБ.
